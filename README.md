# Revenue_Analysis_Data-SQL
Analyzed business revenue data using SQL to calculate key metrics like revenue, profit, cost, and margin. Performed product, regional, and monthly trend analysis to extract actionable insights and support data-driven decision-making.
##  Project Description
Analyzed business revenue data using SQL to calculate key metrics such as revenue, profit, cost, and profit margin. Performed product-wise, region-wise, and monthly trend analysis to extract actionable insights and support data-driven decision-making.

##  Tools Used
- SQL (MySQL)

##  Key Analysis
- Total Revenue, Profit, and Cost calculation
- Profit Margin Analysis
- Product Performance Analysis
- Region-wise Revenue Analysis
- Monthly Sales Trend Analysis
- Online vs Offline Sales Comparison
